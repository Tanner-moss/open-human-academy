# Source Integration Planning Index

This directory tracks source-driven implementation work for every domain and discipline in the academy registry.

## Shared Rule
- Use the full `schools-of-thought.md` section depth instead of shallow seed summaries.
- Major subjects belong in node graphs and skill-tree entry points when they are navigation-sized.
- Finer-grained material belongs in lessons, coursework, labs, projects, and other course detail.
- Rebuild `HumanAcademy.exe` after substantial source-integration passes so the site can be tested immediately.
